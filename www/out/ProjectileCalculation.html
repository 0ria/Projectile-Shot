Constructor

new ProjectileCalculation(velocity, angle, checked, height)

Esta clase calcula los resultados referidos a cada tiro parabólico haciendo uso de los datos de entrada proporcionados por el usuario y pinta la trayectoria en la gráfica.
Parameters:
Name Type Default Description
velocity Number Velocidad inicial que tendrá el proyectil
angle Number Ángulo inicial desde el cuál se lanzará
checked boolean Booleano el cual si está a true hará que el tiro se pinte al canvas
height Number 0 Altura inicial desde la cuál se lanza el proyectil
